# 🎰 룰렛 시스템 사용 가이드

## 개요
Danbi-Bot에 룰렛 시스템이 추가되었습니다. 채팅 메시지에 반응하여 자동으로 룰렛을 실행하고, 유저별 결과를 저장 및 관리할 수 있습니다.

## 주요 기능

### 1. 룰렛 설정
- **최대 4개의 룰렛** 생성 가능
- 각 룰렛마다 **트리거 메시지** 설정
- 룰렛 아이템 추가/수정/삭제
- **확률 합계 100% 검증** (자동 체크)

### 2. 자동 실행
- 채팅에 트리거 메시지가 나타나면 **자동으로 룰렛 실행**
- 확률 기반 랜덤 추첨
- 결과를 채팅으로 자동 출력
- 유저 정보 자동 생성 및 저장

### 3. 유저 관리
- **고유닉(UID)** 자동 추출 및 저장
- 룰렛 결과 히스토리 자동 기록
- DJ가 직접 메모 추가/삭제 가능
- `!킵` 명령어로 유저가 자신의 기록 조회

### 4. 데이터 관리
- 추가된 메모만 선택적으로 다운로드
- JSON 형식으로 내보내기
- 백업/복원 기능 지원

## 사용 방법

### 룰렛 생성하기
1. 메인 화면에서 **"🎰 룰렛 시스템"** 카드의 **설정** 버튼 클릭
2. **"룰렛 관리"** 탭에서 **"+ 새 룰렛 추가"** 버튼 클릭
3. 룰렛 정보 입력:
   - **룰렛 이름**: 예) "감사 룰렛"
   - **트리거 메시지**: 예) "감사합니다."
4. 아이템 추가:
   - **아이템 이름**: 예) "리방하기"
   - **확률 (%)**: 예) 0.5
5. 여러 아이템을 추가하여 **확률 합계가 100%**가 되도록 설정
6. **저장** 버튼 클릭

### 룰렛 실행
- 시청자가 채팅에 트리거 메시지(예: "감사합니다.")를 입력하면 자동 실행
- 결과가 채팅으로 출력됩니다: `🎰 닉네임님의 감사 룰렛 결과: 리방하기`

### 유저 기록 조회
- 시청자가 `!킵` 명령어 입력
- 최근 5개의 룰렛 결과가 채팅으로 출력됩니다

### DJ의 유저 관리
1. 설정 화면에서 **"유저 관리"** 탭 선택
2. 유저 클릭 → **보기** 버튼
3. 유저의 전체 히스토리 확인
4. **추가 메모 작성** 입력란에 메모 입력 후 **메모 추가**
5. 메모 삭제는 각 메모 옆 **삭제** 버튼

### 데이터 다운로드
1. **"유저 관리"** 탭에서 **"💾 추가된 내용만 다운로드"** 버튼 클릭
2. JSON 파일이 다운로드됩니다
3. 파일에는 **추가 메모가 있는 유저**만 포함됩니다

## 설정 예시

### 예시 1: 감사 룰렛
- **룰렛 이름**: 감사 룰렛
- **트리거 메시지**: 감사합니다.
- **아이템**:
  - 리방하기: 0.5%
  - 팬보드 남기기: 1.0%
  - 좋아요 누르기: 5.0%
  - 꽝: 93.5%

### 예시 2: 축하 룰렛
- **룰렛 이름**: 축하 룰렛
- **트리거 메시지**: 축하합니다
- **아이템**:
  - 특별 선물: 10%
  - 일반 선물: 30%
  - 응원 메시지: 60%

## 주의사항
- 확률 합계가 정확히 100%가 아니면 경고 메시지가 표시됩니다
- 최대 4개의 룰렛만 등록 가능합니다
- 트리거 메시지는 정확히 일치해야 작동합니다 (대소문자 구분)
- 고유닉을 가져올 수 없는 경우 닉네임을 UID로 사용합니다

## 데이터 저장 위치
- 룰렛 설정: `rouletteData`
- 유저 정보 및 히스토리: `rouletteData.users`
- 모든 데이터는 Electron Store에 자동 저장됩니다

## 백업 및 복원
- **설정 관리** 섹션의 **"💾 설정 백업 내보내기"** 버튼으로 전체 백업
- **"📂 백업 가져오기"** 버튼으로 복원
- 룰렛 데이터도 백업에 포함됩니다

## 문제 해결
- 룰렛이 작동하지 않으면 **"엔진 재시작"** 버튼 클릭
- 데이터가 저장되지 않으면 **저장** 버튼을 눌렀는지 확인
- 고유닉이 추출되지 않으면 닉네임으로 대체되어 저장됩니다
