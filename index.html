🎰 룰렛 시스템 상세 사용설명서

1. 룰렛 트리거 동작 조건

룰렛은 설정된 '금액에'에 반응하여 자동으로 실행됩니다.

  • 스티커 선물: 트리거 메시지가 "10스푼"인 경우, 시청자가 10스푼을 선물하면 즉시 룰렛이 돌아갑니다.
  • DJ 제외: 무한 루프 방지를 위해 DJ 본인의 채팅에는 룰렛이 반응하지 않습니다. (단, 스티커 선물은 본인도 가능)

2. 유저별 명령어 자동 생성

룰렛 결과가 나오면 해당 유저의 닉네임으로 된 명령어가 자동으로 생성됩니다.

  • 자동 생성: '단비' 유저가 룰렛을 돌리면 !단비 명령어가 명령어 관리 모듈에 추가됩니다.
  • 내용 업데이트: 이미 명령어가 있다면 기존 내용에 새로운 결과를 덧붙입니다.
  • 중복 합산: 동일한 결과가 여러 번 나오면 결과 X횟수(예: 주사위 X4)로 표시됩니다.
  • 공백 처리: 닉네임에 공백이 있어도(예: '단 비') 명령어는 공백 없이(!단비) 생성됩니다.

3. 명령어 사용 방법

  • !닉네임: 유저 본인의 룰렛 결과 목록을 채팅창에 출력합니다.
  • 부분 일치: 닉네임이 길어도 앞 두 글자만 입력하면 실행됩니다. (예: !단비dfdf -> !단비 또는 !단 비로 실행 가능)

4. 설정 팁

룰렛 설정 창에서 아이템별 확률을 조정할 수 있습니다. 확률의 총합은 자동으로 계산되어 공정하게 추첨됩니다.
